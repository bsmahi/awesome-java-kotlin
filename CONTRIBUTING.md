# 🤝 Contributing to Awesome Java & Kotlin

Thank you for considering contributing! 🎉  
This project is a curated list of Java and Kotlin learning resources.

We welcome contributions that help improve the quality and usefulness of this repository.

---

## 📌 What you can contribute

You can contribute:

- ☕ Java learning resources (tutorials, docs, blogs, videos)
- 🟣 Kotlin resources
- 🍃 Spring ecosystem resources
- 🧪 Tools, frameworks, and libraries
- 📚 Interview preparation materials
- 🧠 JVM, performance, and best practices content

---

## 🚫 What NOT to add

Please avoid:

- Low-quality or spam links
- Outdated or broken resources
- Promotional or affiliate-only content
- Duplicate entries

---

## 🛠️ How to contribute

### 1. Fork the repository
Click the **Fork** button on GitHub.

---

### 2. Clone your fork

```bash
git clone https://github.com/<your-username>/awesome-java-kotlin.git
cd awesome-java-kotlin
